# Support

Thanks for using ValorLegend.

## Get Help

- Usage questions: open a GitHub Discussion
- Bug reports: open a GitHub Issue with reproduction steps
- Security concerns: follow `SECURITY.md` and report privately

## Response Expectations

- Best-effort triage for new issues within 3-5 business days
- Security reports are prioritized

## Before Opening an Issue

- Check `README.md` for setup and deployment instructions
- Check existing issues to avoid duplicates
- Include logs, environment details, and exact reproduction steps
